# TV show app

It's a frontend app that use tvmaze api to show series and movies.
 
See demo **[here](https://tv-show-app-eight.vercel.app/)** .


## Installation

You can install the dependencies with these commands.

npm:

```bash
npm install
```

yarn:

```bash
yarn
```

## Deployment

To Start this project run

npm:

```bash
npm run dev
```

yarn:

```bash
yarn dev
```
